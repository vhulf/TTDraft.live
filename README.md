# TTDraft.live source code repository

This is a website meant to provide a new type of gamemode to the DKR community,,, go TTDraft! Want something changed on the site? OR a new setting added? Submit a PR or issue ya goober!

The site was initially made in Next.JS, but has been switched to Vite + React + Tailwind. Newest features implemented with OpenCode because ahhhhhh i guess we gotta use these things now ahhhhhhhhhhhhhhhhh


ANYWAY! Playing TTDraft should be as easy as visiting [TTDraft.live](http://ttdraft.live/)!

Running your own copy of TTDraft should be as easy as:

```
git clone https://github.com/vhulf/TTDraft.live
npm install
npm run dev
```
